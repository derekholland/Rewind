                    //get the values from the Page
// starts or aka controller function
function getValues() {
  // get values from the page
  let startValue = document.getElementById("startValue").value;
  let endValue = document.getElementById("endValue").value;
  let numbers = [];
  // validate input
  // convert input string to an integer
  startValue = parseInt(startValue);
  endValue = parseInt(endValue);
  // check if both variables are integers
  if (Number.isInteger(startValue) && Number.isInteger(endValue)) {
    numbers = generateNumbers(startValue, endValue);
    // call displayNumbers
    displayNumbers(numbers);
  } else {
    alert("You must enter integers");
  }
}

//generate numbers from the start value
// logic function(s)
function generateNumbers(startValue, endValue) {
  let numbers = [];

  // get all numbers from start to end
  for (let i = startValue; i <= endValue; i++) {
    numbers.push(i);
  }
  
  return numbers;
}

// display the numbers and mark even numbers in bold
// display or view functions
function displayNumbers(numbers) {
  let templateRows = "";

  for (let i = 0; i < numbers.length; i++) {
    let className = "even";
    let number = numbers[i];
    if (number % 2 === 0) {
      className = "even";
    } else {
      className = "odd";
    }
    //This does render correctly with Prisim see the source
    templateRows += `${number}`;
  }

  document.getElementById("results").innerHTML = templateRows;
}

The code is structured in multiple functions starting with the getValues() function.

The getValues() Function

The getValues() function accepts no arguments and gets the number input values from the user. The startValue variable is created by using document.getElementById() method to get the value from the input and store this data in the variable. The next variable that is created is called endValue uses the same method to get the value from the endValue attribute of the endValue input tag. Next, the numbers array is created, this will be used later in the program.

Next, the startValue and endValue variables are converted from strings into integers. An if statement is used to check if startValue and endValue variables are integers. If both values are integers two things will happen. The first action that happens is thegenerateNumbers() function is called with the startValue and endValue variables passed as arguments. This function will create an array that has the full range of numbers based on the user's input. Next, the displayNumbers() function is called with the numbers array being passed in as an argument. This final function ends the program by displaying all of the numbers on the page with the even numbers highlighted. In the following functions I will describe how this happens.

The generateNumbers() Function

This function takes in the startValue and endValue variables created in the getValues() function. First, an empty array called numbers is created. Next a for loop is used to get all the numbers starting with the startValue number up to and including the endValue number. During each iteration of the loop the number is pushed into the numbers array. Once the loop completes all iterations and fills the numbers array, the numbers array is returned and since this data is assigned to the numbers variable in the getValues() function, it will be used as an argument when getValues() calls displayNumbers()

The displayNumbers() Function

As the name implies this function is used to display the numbers but it also highlights all of the even numbers. This function accepts one argument and is called in the getValues() function. The argument passed is the array of numbers from the start value to the end value that was created previously in the generateNumbers() function. Next, a for loop runs. This loop runs for the length of the array. During each iteration a variable called className is created and initialized to the value of the string "even". Another variable called number is also created and assigned the value of the current number from the numbers array during that iteration (numbers[i]).

Next an if statement runs to determine if the current number of that iteration is even. If the number is even, the variable className remains assigned the string "even." However, if the number is not even className gets assigned the respective "string." Next, templateRows variable gets the table row and table data with the template literal comprised of the className wedged in between the brackets. The className refers to a custom CSS rule. This his how the even numbers are highlighted. Finally, after the loop creates the table data for each value from the numbers array, the data is embedded into the HTML of the page by setting the innerHTML to the the templateRows variable. The final result is a list of numbers from the range of numbers that the user specified. This gets displayed on the page with all the even numbers highlighted.
